# MediaCompress

# Install

## CLI
- git clone git@github.com:shyjtal/MediaCompress.git
- cd MediaCompress
- ./Install.command

## GUI
- download zip
- upzip MediaCompress.zip
- enter MediaCompress
- double click Install.comman
